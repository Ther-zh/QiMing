# “启明”智能导盲马甲产品设计书（v1.1）

## 一．项目概述

### 1. 项目背景
全球有超过2.5亿视障人士，他们的独立出行是社会面临的普遍性挑战。传统的导盲犬培训成本高、数量有限，而盲杖提供的环境信息极为有限。随着自动驾驶、边缘计算和人工智能技术的成熟，我们有机会将这些前沿技术集成到可穿戴设备中，为视障人士创造一个全天候、高精度的“AI导盲犬”，从根本上提升他们的行动自由与生活质量。

### 2. 产品简介
“启明”是一款基于多模态传感器融合与边缘AI计算的智能导盲马甲。它通过模仿自动驾驶系统的环境感知、决策规划和控制执行流程，为穿戴者提供实时、精准的导航与避障指引。产品以非侵入式的触觉震动和自然语音作为主要交互方式，旨在成为视障人士可靠、贴心的出行伙伴。

### 3. 项目目标与创新点

**总体目标：** 开发一款安全、可靠、易用的智能穿戴导盲设备，提升视障人士的出行安全性、独立性和生活质量。

**核心功能目标：**
1. **精准环境感知：** 实现360°范围内的障碍物（静态/动态）检测与追踪、坑洼识别、交通信号灯识别、人行道边界识别等。
2. **智能路径规划：** 结合高精度地图与实时感知，进行安全路径规划与导航。
3. **自然的人机交互：** 通过3D空间音频和分布式微震动阵列，提供直观、不刺耳的导航指令。
4. **边缘实时处理：** 所有核心计算在本地完成，确保低延迟、高可靠性，并保护用户隐私。

**核心创新点：**
1. **技术融合创新：** 将L4级自动驾驶的“感知-决策-执行”技术栈，首次微型化并集成于可穿戴马甲中。
2. **交互模式创新：** 结合语音宏观指令与触觉微观指引。语音用于告知“去哪里”，触觉用于实时指导“如何走”，避免“语音疲劳”和在嘈杂环境中的信息丢失。
3. **架构创新：** 基于边缘计算，打造高可靠性、低延迟、数据隐私安全的系统，不依赖云端网络，实现全天候稳定工作。

## 二．市场调研与分析

### 1. 目标市场
- **主要用户：** 全球的中度至全盲视障人士。
- **潜在用户：** 在复杂室内环境（如医院、大型商场）中需要辅助导航的老年人。

### 2. 市场现状与痛点

**现有方案：**
- **盲杖：** 技术简单廉价，但探测范围极小（仅身体前方），无法探测空中（如低矮树枝）和远处障碍。
- **导盲犬：** 机动性好，但培训成本高、周期长，数量稀缺，且需要用户具备与犬只配合的能力。
- **电子导盲杖/头戴设备：** 功能单一，用户体验不佳（如需要手持、形态引人注目），多数产品仍处于实验室阶段，未能很好解决实际复杂场景问题。

**核心痛点：**
- **安全性不足：** 对突然出现的移动物体（如自行车、汽车）、地面坑洼无法有效预警。
- **信息维度少：** 无法识别红绿灯、公交站牌等语义信息
- **不够自然与独立：** 现有设备交互方式笨拙，无法让用户“忘记设备存在”，自然地融入环境

### 3. 竞争格局
目前市场上暂无将多模态传感器与边缘AI计算以马甲形态深度融合的成熟竞品。我们的产品有很多市场优势：三维、全景、语义级的环境感知，采取边缘计算拥有高可靠性，可穿戴设备。

### 4. 产品优势

1. **全景、精准的环境感知能力，超越传统工具**
   - 通过融合多种传感器（摄像头、固态激光雷达、IMU），实现了360°无死角的环境探测。
   - 能够识别多维度的障碍物，包括静态障碍、动态车辆、低矮的坑洼，甚至语义信息（如交通信号灯、人行道边界），彻底解决了盲杖探测范围小、信息量少的根本痛点。

2. **基于边缘计算的实时性与高可靠性**
   - 所有核心计算在本地边缘设备完成，带来了极低的处理延迟，确保了避障和导航指令的即时性，对于保障用户安全至关重要。
   - 不依赖云端网络，可在任何网络环境（甚至无网）下稳定工作，避免了因网络延迟或中断带来的安全风险，实现了真正的全天候运行。

3. **创新、自然且不扰民的人机交互模式**
   - 创造性地采用了 “语音宏观指令”的模式交互。
   - 语音用于告知目的地和宏观路况，清晰直接；分布式触觉震动用于实时、无声的转向和避障指引，避免了在嘈杂环境中语音指令被淹没，也减少了长时间使用带来的“语音疲劳”，让交互更自然、不引人注目。

4. **人性化的工业设计与全天候适用性**
   - 采用符合人体工学的马甲形态，解放了用户的双手，相比需要手持的设备（如电子导盲杖）更具便利性和灵活性。
   - 轻便透气的材质和模块化设计保障了穿戴舒适度，IP67级的防水防尘和长续航能力使其能够应对各种日常及户外出行场景。

## 三. 产品详细设计

### 1. 系统架构（分层设计）

**感知层：**
- **主视觉模块：** 前向主摄像头RGB，斜向下的结构光深度相机，用于识别远距离交通信号、车辆、行人。左右后侧采用鱼眼摄像头。
- **固态激光雷达：** 用于探测远处移动物体（如汽车）的速度和距离。
- **惯性测量单元：** 用于追踪用户的身体朝向和步态。
- **麦克风：** 语音控制。

**边缘计算层：**
- **硬件：** 搭载高性能、低功耗的边缘AI计算芯片（如NVIDIA Jetson Orin Nano系列），集成于马甲背部，内置散热系统。
- **软件算法栈：**
    1. **多传感器融合算法：** 融合视觉、LiDAR、雷达数据，生成统一的、冗余的环境模型。
    2. **轻量化AI模型：**
        - 目标检测与识别：(YOLO系列、EfficientDet等) 识别各类障碍物、交通设施
        - 语义分割：(DeepLab系列) 精确划分可通行区域、人行道、草地等
        - SLAM：实现即时定位与地图构建，结合预置高精度地图进行辅助定位。
    3. **决策规划引擎：**
        - 大模型决策
        - 将以上模型所处理后的数据统一至大模型，训练大模型根据现有情况给出最安全合理的决策，并以适当语言进行提示。

**应用层（人机交互）：**
- **语音交互系统：**
    - 语音输出（TTS）：用于播报宏观信息，如“前方100米到达地铁站A口”、“现在是红灯，请等待”。
    - 输入（ASR）：集成大模型（如优化后的端侧LLM），支持自然语言命令，如“带我去最近的公交站”、“寻找路边的长椅”。
- **紧急警报系统：** 当检测到极高风险（如车辆快速接近）时，系统将启动语音和强烈震动组合警报。

### 2. 工业设计与可穿戴性
- **材质：** 采用透气、轻便、耐磨的户外面料，内置柔软的缓冲层保护电子设备。
- **结构：** 符合人体工学的剪裁，保证运动灵活性。传感器模块采用模块化设计，便于更换和升级。
- **防水防尘：** 达到IP67等级，应对各种天气。
- **充电与续航：** 可拆卸大容量电池组，支持快充，目标续航时间 > 8小时。

## 四. 技术方案

### 1. 关键技术
1. **多模态传感器融合技术：** 解决不同传感器数据在时间和空间上的同步与对齐问题，实现优势互补。
2. **轻量化边缘AI模型部署：** 对模型进行剪枝、量化、知识蒸馏，在保证精度的前提下，满足边缘设备的算力和功耗约束。
3. **基于触觉的引导算法：** 研究震动频率、强度、模式与导航指令的映射关系，确保指引直观无误。
4. **端侧大模型优化：** 将大语言模型轻量化并部署于端侧，实现快速、隐私安全的语音交互。

### 2. 潜在风险与对策
- **风险1：** 极端天气下传感器性能下降。
  - **对策：** 多传感器冗余设计（如雷达在雨雾天补足视觉），并开发针对性的数据清洗和补偿算法。
- **风险2：** 计算功耗与散热挑战。
  - **对策：** 采用先进的低功耗芯片，优化算法效率，设计被动与主动结合的散热风道。
- **风险3：** 用户接受度和信任度。
  - **对策：** 与视障协会合作，从设计初期就让视障人士参与体验和迭代，建立严格的安全测试标准和缓慢的引导训练模式。
